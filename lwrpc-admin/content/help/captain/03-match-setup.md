# Match Setup

Match Setup lets captains choose the planned doubles teams for an upcoming match.

![Match Setup screen](/help/captain/screenshots/match-setup.png "Match Setup screen")

## Complete every line

Every listed doubles team must have two players before Match Setup can be saved. The system checks for duplicate players and team rating limits before allowing the save.

## Rating requirements

Players with **Rating Needed** can remain on the roster, but they are not available for Match Setup until the selected rating type has a valid value.

## Email players

After Match Setup is saved, use **Email Players** to send the lineup to selected players or all roster players with email addresses.

## Match score sheets

Use **Match Score Sheet** from the upcoming match card to preview, download, or print the paper score sheet for the match.

![Match score sheet](/help/captain/screenshots/match-scoresheet.png "Match score sheet")

# Score Verification

Pending score verification appears when the opposing captain has submitted scores for your match.

![Pending Score Verification](/help/captain/screenshots/pending-score-verification.png "Pending Score Verification")

## Validate scores

Open the match, review the line players and game scores, then use **Validate Scores** when everything is correct.

## Dispute scores

Use **Dispute Scores** when a correction is needed. Add clear notes so the submitting captain knows what to fix.

## Final standings update

Verified scores update standings and match history. Final bye adjustments are applied when the match is part of a division schedule that uses bye scoring.
